# raspberry-pi-car
# 🚗 Raspberry Pi Car Control (Keyboard Controlled)

This project allows you to control a 2WD robot car using a keyboard in real-time.

## 🎮 Controls

* W → Forward
* S → Backward
* A → Left
* D → Right
* Release → Stop

## ⚙️ Components Used

* Raspberry Pi
* L298N Motor Driver
* 4 DC Motors
* Battery

## 🧠 Features

* Real-time keyboard control using pygame
* PWM speed control
* Simple and clean GPIO setup

## 🔧 Installation

```bash
pip3 install -r requirements.txt
```

## ▶️ Run

```bash
python3 car_control.py
```
